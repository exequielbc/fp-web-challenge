## Getting Started

Run the development server using:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Live deployment

This app is deployed in [https://fp-web-challenge.vercel.app/](https://fp-web-challenge.vercel.app/)
